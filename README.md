# 📈 StockVision AI - Intelligent Stock Market Analysis & Prediction

StockVision AI is a Machine Learning-based web application that analyzes historical stock market data and predicts future stock trends. The project leverages data science, visualization, and predictive modeling techniques to help users gain insights into stock performance and make data-driven decisions.

---

## 🚀 Features

- 📊 Historical Stock Data Analysis
- 📈 Interactive Stock Price Visualization
- 🤖 Machine Learning Based Price Prediction
- 🔍 Exploratory Data Analysis (EDA)
- 📉 Trend Analysis using Moving Averages
- 🌐 User-Friendly Web Interface with Streamlit
- 📅 Custom Stock Selection and Date Range Analysis
- 📥 Real-Time Data Fetching from Yahoo Finance

## 📊 Dataset

The project uses stock market data obtained from Yahoo Finance.

### Data Source
🌐 https://finance.yahoo.com/

### Dataset Attributes

| Feature | Description |
|----------|------------|
| Date | Trading date |
| Open | Opening stock price |
| High | Highest stock price during the trading session |
| Low | Lowest stock price during the trading session |
| Close | Closing stock price |
| Volume | Number of shares traded |
